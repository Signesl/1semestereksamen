Grundlæggende Web

Projektløsning

Implementering af HTML, CSS og responsive designprincipper

I dette tema blev vi introduceret til grundlæggende HTML og CSS. Vi arbejdede i Visual Studio Code, hvor vi implementerede udleveret tekstindhold i vores HTML-filer samt fulgte wireframes for at sikre korrekt struktur og opbygning af layoutet. Opgaven fungerede samtidig som en del af studiestartsprøven.

Opgaven bestod blandt andet i at udvikle et responsivt website, der fungerer optimalt på både mobil og desktop. Derudover blev vi introduceret til vigtigheden af designprincipper som gestaltprincipper, styletiles, typografi og tekstopsætning, som tilsammen bidrager til et sammenhængende og brugervenligt design.

Vi arbejdede også med billedformater og lærte at vælge de rette filtyper i forhold til kvalitet, ydeevne og anvendelse på web.

Projektproces

I opgaven arbejdede jeg ud fra udleverede wireframes og layoutdiagrammer, som dannede grundlag for struktur og opbygning af hjemmesiden. Jeg implementerede tekstindholdet i HTML-filer i Visual Studio Code og anvendte CSS Grid og Flexbox til at opbygge layoutet.

Herefter stylede og tilpassede jeg hjemmesiden individuelt gennem CSS og arbejdede med farver, typografi og visuelle elementer. Som en del af processen optimerede jeg de udleverede billeder ved at konvertere dem til WebP-format og reducere filstørrelsen uden at gå på kompromis med kvaliteten.

For at sikre et responsivt design anvendte jeg media queries i CSS, så layoutet tilpasser sig forskellige skærmstørrelser. Den færdige løsning blev afleveret som en del af studiestartsprøven.

Erfaringer og indsigter

Gennem temaet blev jeg introduceret til fundamentet for webudvikling gennem HTML og CSS samt til centrale designkonventioner og gestaltprincipper, som er vigtige for at skabe brugervenlige og overskuelige websites. Jeg opnåede en grundlæggende forståelse for, hvordan struktur, layout og visuelt hierarki hænger sammen.

I arbejdet med opgaven eksperimenterede jeg med farver og styling direkte på sitet uden først at dokumentere mine designvalg, hvilket gav mig en vigtig læring. Denne erfaring tog jeg med videre til de efterfølgende temaer, hvor jeg arbejdede mere struktureret med planlægning og dokumentation før implementering.

Derudover lærte jeg om billedformater og billedoptimering, herunder hvornår man anvender JPEG, PNG, SVG og WebP, og hvordan valget af filformat påvirker både kvalitet, performance og brugeroplevelse. Denne viden er central i arbejdet med at optimere websites effektivt og bæredygtigt.

Samlet set gav temaet mig praktisk erfaring med hele processen fra grundlæggende HTML og CSS til færdigt responsivt design samt en introduktion til wireframes og mockups i Figma.
